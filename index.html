Guía Rapida de Comandos Git

Git init

Inicializa un nuevo repositorio Git local en el nuevo directorio.

Git add .

Agregar todos los archivos al área de preparacion (Staging).

Git push

Envia los comitslocales al repositorio remoto.

Git estatus

Muestra el estado actual del repositorio archivos modificados y no rastreados.

Git commit -m "mensaje"

Guarda los cambios agrgados con un mensaje descriptivo

Git pull

Obtiene y fuciona cambios del repositorio remoto .
